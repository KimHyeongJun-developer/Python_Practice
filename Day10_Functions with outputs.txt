독스트링
함수나 다른 블록을 만들 때 작은 문서조각을 만드는 것.
큰따옴표 3개(""") 로 설정할 수 있으며, 내가 정의한 함수에 대한 간략한 설명을 적는 곳이다.

함수를 변수에 저장할려면 괄호와 매개변수는 사용하지 않는다. 이는 딕셔너리에서도 마찬가지이다.

def add(n1, n2):
    return n1 + n2

def multiply(n1,n2) :
    return n1 * n2

def subtract(n1, n2) :
    return n1 - n2

def divide(n1, n2) :
    return n1 / n2

operation = {
    "+" : add,
    "*" : multiply,
    "-" : subtract,
    "/" : divide
}

다음과 같은 함수와, 딕셔너리를 지정했다고 하자. 우리는 딕셔너리의 키를 통해서 함수에 접근할 수 있고,
함수의 매개변수를 지정해주기 위해서는
operation["+"](1,2) 와 같은 형태로 접근할 수 있따.
